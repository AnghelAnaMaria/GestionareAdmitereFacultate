# GestionareAdmitereFacultate
Spring Boot Project

## Descriere
Proiectul se ocupa de procesul de admitere, permițând gestionarea candidaților, facultăților și examenelor. Proiectul urmărește să faciliteze înscrierea candidaților, monitorizarea examenelor și calcularea rezultatelor.

## Business Requirements
1. Candidatul se poate înregistra în sistem cu datele personale.  
2. Candidatul se poate înscrie la unul sau mai multe examene.  
3. Fiecare facultate are un număr limitat de locuri disponibile.  
4. Se poate calcula nota finală pentru admitere.  
5. Administratorul poate adăuga/modifica facultăți și examene.  
6. Candidatul poate vedea rezultatele examenelor.  
7. Sistemul trimite notificări candidaților despre programări.  
8. Se poate genera raport cu lista candidaților admiși.  
9. Sistemul validează că un candidat nu se poate înscrie la examene duplicate.  
10. Sistemul menține istoricul înscrierilor și rezultatelor pentru fiecare candidat.

## MVP Features
1. **Gestionare candidați** – Adăugarea, modificarea și vizualizarea candidaților în sistem.  
2. **Gestionare facultăți** – Crearea și modificarea facultăților și a locurilor disponibile.  
3. **Înscriere la examene** – Candidații se pot înscrie la examene și sistemul validează disponibilitatea.  
4. **Calculul rezultatelor** – Sistemul calculează nota finală și stabilește dacă candidatul este admis.  
5. **Vizualizare rezultate** – Candidații și administratorii pot vedea rezultatele examenelor.
